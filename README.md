## üìñ About

This project is a **Square Equation Calculator** that finds the roots of a quadratic equation of the form:

\[
ax^2 + bx + c = 0
\]

It accepts coefficients `a`, `b`, and `c` from a file and computes the roots using the quadratic formula.
